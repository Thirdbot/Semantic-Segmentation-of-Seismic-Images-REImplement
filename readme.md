# Implementation of Semantic Segmentation Of Seismic Image

## from: https://arxiv.org/pdf/1905.04307
## using: rockML
## env: Conda

## support only: tensorflow >= 2.10 <= 2.16

# Installations
```bash
conda env create -f environment.yml

. activate rockml

git clone git@github.com:IBM/rockml.git
```

# test installations
```bash
cd rockml

pytest tests
```
